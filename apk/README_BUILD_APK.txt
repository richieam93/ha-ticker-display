Die mitgelieferte alte APK wurde entfernt, damit nicht versehentlich eine veraltete Version installiert wird.

Neue APK bauen:

cd tickerdisplay_app
./gradlew assembleRelease

Die erzeugte APK liegt danach unter:
app/build/outputs/apk/release/

Für einen schnellen Test kannst du auch verwenden:
./gradlew assembleDebug
